XML document handling functions

The XML document handling functions can be used to read and write JSON format data.

The names of the XML element tags (node names) added do not need to conform to XML naming conventions/restrictions if only JSON is being generated.

Blank JSON label names are supported.

The XML functions support parsing and generating a JSON root array. When parsing a root array, the root node is named "root" and its child array nodes are named "element".

To generate a root array, the XML-SET-JSON-ARRAY function must be called with a parameter specifying the root node. A root array is not generated if the root node contains anything other than a single array.

Pronto RAD supports the following XML document handling functions:

Document functions

  • XML-CLOSE-DOCUMENT()

    Closes an internal XML document

  • XML-DOCUMENT-IS-JSON()

    Returns 1 if the document is in JSON format, otherwise returns 0

  • XML-NEW-DOCUMENT()

    Creates a new internal XML document and returns a handle to the document

  • XML-SAVE-AS-FILE()

    Saves an internal XML document to a file

  • XML-SAVE-AS-FILE-EX()

    Saves an internal XML document to a file

  • XML-SAVE-AS-TEXT()

    Saves an internal XML document to the specified text field

  • XML-SAVE-AS-TEXT-EX()

    Saves an internal XML document to the specified text field

  • XML-VALIDATE-DOC()

    Validates a document against the specified XSD

Parsing XML

  • XML-PARSE-FILE()

    Parses the contents of a specified XML file and returns a handle to the internal XML document

  • XML-PARSE-TEXT()

    Parses the contents of a text field containing the XML text and returns a handle to the internal XML document

Freeing or copying node handles

  • XML-COPY-NODE-HANDLE()

    Returns another handle to the specified node

  • XML-FREE-NODE-HANDLE()

    Frees the node handle and deallocates the resources associated with that handle

Allocating and returning node handles (after parsing an XML request)

  • XML-CHILD-NODE-TEXT()

    Adds a new child text node of the name and namespace specified to the end of the child node list of the parent

  • XML-GET-CHILD-BY-NAME()

    Allocates and returns a handle to the first child node of the parent that matches the specified name

  • XML-GET-DOC-ENCODING()

    Returns a string containing the encoding of a specified XML document

  • XML-GET-FIRST-ATTRIBUTE-NAME()

    Returns the name of the first attribute/property in the list of attributes associated with the specified node

  • XML-GET-FIRST-CHILD-NODE()

    Allocates and returns a handle to the first child node of the specified parent node

  • XML-GET-LAST-CHILD-NODE()

    Allocates and returns a handle to the last child node of the specified parent node

  • XML-GET-NEXT-ATTRIBUTE-NAME()

    Returns the name of the next attribute/property in the list of attributes associated with the specified node

  • XML-GET-NEXT-NODE()

    Sets the specified node handle to point to the next node in the list

  • XML-GET-NS-PREFIX-URL()

    Returns the URL associated with the specified namespace prefix

  • XML-GET-NODE-ATTRIBUTE()

    Returns the text value of a specified attribute, or SPACES if the attribute is not defined for the specified node

  • XML-GET-PREV-NODE()

    Sets the specified node handle to point to the previous node in the list

  • XML-GET-ROOT-NODE()

    Allocates and returns a handle to the root node of an XML document

  • XML-NEXT-ELEMENT-SIBLING()

    Sets the specified node handle to point to the next node in the list of the same type

  • XML-NODE-NAME()

    Returns a string containing the name of the specified node

  • XML-NODE-NS-PREFIX()

    Returns a string containing the prefix of the name space of the specified node, or SPACES for the default name space

  • XML-NODE-NS-URL()

    Returns a string containing the URL or HREF associated with the name space

  • XML-NODE-STRING()

    Returns a text string comprising the text nodes and entity references contained in the node

  • XML-NODE-TEXT()

    Returns the text value of the content of the specified node

  • XML-NODE-TYPE()

    Returns the numeric type of the specified node

  • XML-PREV-ELEMENT-SIBLING()

    Sets the specified node handle to point to the previous node in the list of the same type

  • XML-SET-JSON-ARRAY()

    Treat the node as an array if JSON is produced

Adding node elements to an XML response

  • XML-ADD-CHILD-NODE()

    Adds a new child element node of the specified name to the end of the child node list of the specified parent node

  • XML-ADD-CHILD-NODE-TEXT()

    Adds a new child text node of the name and namespace specified to the end of the child node list of the parent

  • XML-ADD-NODE-ATTRIBUTE()

    Adds an attribute of the specified name and value to the specified node

  • XML-ADD-NS()

    Adds a name space to the specified node for use by child nodes

  • XML-ADD-CHILD-NODE-BLOB()

    Adds a new child text node of the name and namespace specified to the end of the child node list of the parent

  • XML-ADD-CHILD-NODE-CLOB()

    Adds a new child text node of the name and namespace specified to the end of the child node list of the parent

  • XML-ADD-CHILD-NODE-NO-QUOTES()

    Adds a new child text node of the name and namespace specified to the end of the child node list of the parent

  • XML-ADD-CHILD-NODE-NUMBER()

    Creates a child node containing the numeric value specified in <num-expr>, formatted with the number of decimal places specified

Deleting or modifying node content

  • XML-DELETE-NODE()

    Deletes the specified node

  • XML-DELETE-NODE-ATTRIBUTE()

    Deletes the specified attribute from the specified node

  • XML-MODIFY-NODE-ATTRIBUTE()

    Modifies the value of the specified attribute

  • XML-MODIFY-NODE-TEXT()

    Modifies the text content stored by the specified node
