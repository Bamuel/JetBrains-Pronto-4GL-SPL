Data definition picture strings

The following picture string characters are used to specify the format of a field being defined. They indicate the size of the field, number of integer/decimal places, and whether the field allows negative amounts.

The following is a list of the valid picture string characters and their use.

Picture Strings
Characters Description

9

Represents a single numeric digit in the range 0 to 9.

When this character appears after a decimal point, it represents a decimal digit; otherwise, it represents an integer digit.

Z

Similar to 9 except leading zeros are suppressed (left blank).

*

Similar to 9 except leading zeros are replaced with an *.

$

Display the currency sign in the amount.

When this character is repeated multiple times, the currency sign will float over leading zeros to be positioned before the first non-zero digit.

. (Dot) or V

Specifies the location of the decimal point.

This separates the integer places from the decimal places. If this is not specified then no decimal places is displayed.

- (Minus) or S

Indicates the position of the sign character when a value is negative.

This character, when used, must appear at the start of the picture string. When this character is repeated multiple times, the minus sign floats over leading zeros to be positioned before the first non-zero digit.

T

Similar to . except all decimal places and the decimal point are shown as blank when the value is zero.

/

Insert / symbol.

B

Insert blank character (space).

A space can also be specified, however, the picture string must be enclosed in quote characters.

%

Inserts a % sign.

,

Insert , symbol if significant digits already found, otherwise leave blank.

DR or CR

If the amount is negative then insert DR or CR, otherwise insert two blanks.

We do not recommend that you use these, as these symbols are only meaningful in English-speaking countries.

()

Displays a negative value in brackets.

The bracket characters must surround all other picture string characters.

X

Represents a single alphanumeric character.

This is for defining alphanumeric fields only and should not be combined with any of the above characters.

ddmmmyybhh:mm

Specifies the data and time.

display lf-date-time @4,10 pic ddmmmyybhh:mm

will display the date and time as 10Oct60 01:22

where:

dd

day number

mmm

month name

yy or yyyy

year

b

blank character

hh

hour

mm

minute

ss

second

Certain picture characters are often required to be repeated.

To specify a picture of ten alphanumeric characters enter:

XXXXXXXXXX

Another way of specifying the repetition of a picture character is by using repetition factors. The repetition of a picture character can be expressed as follows:

pic-char(amount)

so

x(10)

The PIC clause is also used as Data Edit strings on the DISPLAY and ACCEPT statements. For more information see the COL clause of the DISPLAY statement.
