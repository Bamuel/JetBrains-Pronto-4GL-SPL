Software Development Kit (SDK)

Did you find this topic while searching for dictionary related information? Please see the separate SDK documentation as described below.

The Software Development Kit documentation is supplied as a separate help file that can be installed by running the documentation installer in the doc directory of the Pronto Xi installation media.

The SDK provides specialised assistance for customising, integrating or extending Pronto Xi as part of an implementation project. The SDK contains detailed technical information, including:

  • Pronto Object ActiveX control

  • Component libraries

  • API calls

  • Program calling methods

  • Suppressed data entry fields

  • Dictionary objects and fields

  • Code samples
